Marginalia Query Service

This is the barebones search interface for the Marginalia Search Engine software.

If you are seeing this somewhere other than your own machine, someone has likely misconfigured something.

Control GUI

If this is running on your own machine, you can also use the control interface, which is by default available on http://localhost:8081/.

API

The /search endpoint offers a web interface for performing queries. The same endpoint is available as a JSON API when the Accept: application/json header is sent by the client.

Query parameters

ParameterDefaultDescription
query or q (required) The search query string.
count 20 Number of results (max 100).
dc 2 Max results per domain (1-100).
timeout 150 Query timeout in milliseconds (50-250).
filter NO_FILTER Named system filter, e.g. POPULAR, SMALLWEB, BLOGOSPHERE, VINTAGE, TILDE, ACADEMIA, PLAIN_TEXT, FOOD, FORUM, WIKI, DOCS.
nsfw 1 NSFW filter tier: 0=off, 1=danger, 2=danger+porn
page 1 Page number for pagination.
lang en Language ISO code.

Minimal integration with the API:

Curl
$ curl -H 'Accept: application/json' 'http://localhost:8080/search?query=test&count=20'
Python 3
#!/bin/env python3 import requests import json r = requests.get('http://localhost:8080/search?query=test', headers={'Accept': 'application/json'}) print(json.dumps(r.json(), indent=4))

Remember to URL-encode the query!
